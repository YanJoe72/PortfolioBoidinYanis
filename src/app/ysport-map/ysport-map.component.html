Projet YSport :

Contexte :

Au cours de notre programme académique en informatique, nous avons été chargés de concevoir et de développer une application web en utilisant les frameworks React et Spring Boot.

Aperçu :

Description :

Le BTS SIO dispose d’une bibliothèque de revues informatiques. Pour faciliter la gestion de cette bibliothèque, nous souhaitons développer une application permettant de gérer l'approvisionnement des revues ainsi que le suivi des emprunts et des retours. À chaque étape, à partir de la description des tâches, il sera nécessaire de modéliser les données avec un modèle entité-association (MEA). L’application devra ainsi permettre :

Description :

Le but de ce projet est de réaliser un site web avec une carte interactive qui renseigne 28000 complexes sportifs. Les utilisateurs peuvent avoir des renseignements sur les complexes grâce à la map. S'il cherche un complexe en particulier, une option de filtrage a été mise en place.

Technologies :

React+Vite pour le front et Spring Boot pour le back.
