Projet Générateur SQL :

Contexte :

Au cours de notre programme académique en informatique, nous avons été chargés de concevoir et de développer un générateur SQL en utilisant le langage de programmation C#. Ce projet visait à nous immerger dans le domaine de la manipulation de données relationnelles tout en nous permettant d'acquérir une expérience pratique dans le développement logiciel.

Aperçu :

Description :

Notre objectif principal était de créer un générateur SQL en C# qui prendrait en entrée des fichiers texte au format spécifique. Ces fichiers contenaient des données sous la forme : id;nom;prenom. Le générateur devait alors transformer ces données en requêtes SQL de type INSERT, facilitant ainsi l'insertion de ces données dans une base de données relationnelle.
