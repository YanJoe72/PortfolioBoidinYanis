Projet BioRelai :

Contexte :

Le but de ce projet était de réaliser une connexion et une inscription pour des producteurs. Le site web a été réaliser en PHP.

Aperçu :

Description :

Quand le producteur se connecte, il a la possibilité d'ajouter des nouveaux produits, il peut aussi les propose à la vente. Une inscription est aussi possible pour les nouveaux producteurs.

Technologies :

HTML/CSS, PHP et MySQL.
