Stage dans l'entreprise C-monetiquette :

Présentation :

L’entreprise dans laquelle j’effectue mon stage se nomme c-monetiquette. Cette dernière a pour activité la vente d'étiquettes personnalisées par le biais d’un site internet. Cette personnalisation passe par leur site internet. Cela peut aller de simples étiquettes à des gourdes personnalisées ou même des lunchbox. L’entreprise est en service depuis 2013. Son activité s'étend sur la France, la Belgique, le Royaume-Uni, l’Italie et l’Espagne. Chacun de ces pays possède un site propre traduit dans leur langue. L’entreprise c-monetiquette possède une vingtaine de salariés. Actuellement je suis dans le service informatique, plus précisément le service de développement. Ce service possède cinq employés. Une partie s’occupe de la liaison entre la base de données et les machines qui permettent l’impression des étiquettes. L’autre partie s’occupe de l’implémentation des nouvelles fonctionnalités pour le site ou pour directement les salariés. La principale technologie utilisée par les développeurs est Magento. Beaucoup de fonctionnalités sont disponibles par le biais de Magento, mais la principale est la fonction de base de données. Magento est utile pour les développeurs comme pour les autres employées car elle permet de visualiser les différentes statistiques grâce à une interface.

Objectifs :

L’entreprise utilise différents référencement de statistique, tels que Google Ads, META, Microsoft Bing, etc… Leurs données se trouvent dispersées sur différents sites, en plus des données fournies par Magento. En plus de cela, les données ne sont pas agréables à visualiser car l’interface n’est pas accommodée à l’entreprise. Donc un besoin c’est créer celui de réunir ces données et de les rendre plus digestes, pour les fournir aux employées. Pour cela nous utiliserons l’outil Power BI de Microsoft à la demande de l’entreprise. Pour répondre à cette problématique j’utiliserai PHP et SQL pour récupérer ces données pour permettre de les afficher sur Power BI. Actuellement je ne suis pas sur des délais du projet, car je ne me suis pas encore accommoder des différentes technologies utilisées. Ce projet sera réalisé seul.

Technologies Utilisées :

Pendant ce projet, j'ai utilisé plusieurs technologies pour collecter, stocker et visualiser des données. J'ai commencé par utiliser PHP pour le web scraping, permettant ainsi de récupérer les données à partir de différentes sources en ligne. Ensuite, j'ai utilisé SQL pour concevoir et gérer une base de données afin de stocker ces données de manière structurée et efficace.

Une fois les données collectées et stockées, j'ai utilisé Power BI pour les transformer en visualisations interactives et informatives. Power BI m'a permis de créer des tableaux de bord dynamiques et des rapports visuels, offrant ainsi une analyse approfondie et une compréhension claire des informations extraites. Cette approche intégrée de collecte, stockage et visualisation des données a été essentielle pour obtenir des insights précieux et prendre des décisions éclairées dans le cadre de ce projet.
