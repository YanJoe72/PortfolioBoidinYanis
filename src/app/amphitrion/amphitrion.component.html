Projet Amphitryon :

Contexte :

Dans le cadre de notre programme scolaire, nous avons entrepris un projet visant à développer une application mobile pour la gestion d'un restaurant. Réalisée sur Android Studio en Java pour la partie mobile et en PHP pour le back-end, cette application a pour but de simplifier et d'améliorer la gestion des commandes, des stocks et des réservations, tout en offrant une interface intuitive pour les serveurs, cuisiniers et le gestionnaire de salle.

Aperçu :

Description :

Notre objectif principal était de réaliser 3 parties : le cuisinier qui va pouvoir modifier le menu du jour, il va pouvoir ajouter ou supprimer des aliments. Le gestionnaire de salle va pouvoir modifier l'emplacement des tables et pour finir le serveur qui va pouvoir noter les commandes associées à chaque table. Pour ce projet je me suis occuper du cuisinier.

Technologies :

Java pour l'application mobile et PHP pour le back.
