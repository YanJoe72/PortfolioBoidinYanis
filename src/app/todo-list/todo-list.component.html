Projet TodoList :

Contexte :

Le projet TodoList Manager est une initiative développée pour répondre aux besoins de gestion de tâches et de listes de tâches au sein d'une organisation. L'objectif principal est de créer une application web moderne et efficace qui permettra aux utilisateurs de créer, organiser et suivre leurs tâches de manière transparente. Le développement de cette application se fait en utilisant Angular pour le côté client et Spring Boot pour la création d'une API robuste côté serveur.

Description :

TodoList Manager est une application web de gestion de tâches permettant aux utilisateurs de créer des listes de tâches, d'ajouter des tâches à ces listes, de les marquer comme terminées et de les organiser selon leurs priorités. L'application est conçue avec une interface utilisateur conviviale et réactive grâce à Angular côté client, offrant ainsi une expérience utilisateur moderne et intuitive.

Technologies utilisées :

Frameworks : Angular et Spring Boot.

Langages : MySQL, HTML, CSS, TypeScript, Java.
