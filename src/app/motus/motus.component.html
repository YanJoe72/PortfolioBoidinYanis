Projet Motus :

Contexte :

Le projet consiste en la création d'une version du jeu Motus en utilisant le langage C#. Contrairement à une approche orientée objet, où les données et les fonctionnalités sont encapsulées dans des objets, ce projet utilise probablement une approche procédurale ou fonctionnelle, où les opérations sont réalisées de manière séquentielle et basées sur des fonctions ou des procédures.

Aperçu :

Description :

Le projet consiste en la création d'une version du jeu Motus en utilisant le langage C#. Le programme récupère un fichier contenant une liste de mots, puis en choisit un au hasard comme mot mystère. L'utilisateur est ensuite invité à proposer des mots de la même longueur que le mot mystère.

Génération de mots : Le jeu utilise un fichier de mots pour sélectionner un mot mystère aléatoire.

Interface utilisateur : Le jeu offre une interface utilisateur conviviale où les utilisateurs peuvent saisir leurs propositions de mots.

Comparaison de lettres : Le programme compare les lettres du mot proposé par l'utilisateur avec celles du mot mystère.

Indications visuelles : Les lettres sont colorées pour indiquer si elles sont correctes (en vert et à la bonne place), incorrectes (en rouge) ou correctes mais mal placées (en orange).
